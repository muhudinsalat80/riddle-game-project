Riddle Quest – The Enigma’s Edge

Riddle Quest is a simple game built with React where players solve riddles and earn points by choosing the correct answers. The game limits the number of attempts and ends when all tries are used.

Features

Multiple riddles

Correct answers highlighted

Wrong answers highlighted

Limited number of tries

Score and level system

Try again loads a different riddle

Game over screen with results

Technologies Used

React

JavaScript

HTML

CSS

Installation

Clone the repository:

git clone https://github.com/aminSHARIFF/riddle-game-project.git


Navigate into the project folder:

cd riddle-game-project/riddle-game


Install dependencies:

npm install


Run the project:

npm start

How to Play

Select the correct answer from the given choices. If the answer is correct, you gain points and move to the next riddle. If the answer is wrong, your tries decrease. When tries reach zero, the game ends and shows the final results.
